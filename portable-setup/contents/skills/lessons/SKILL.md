# AI Training Curriculum

A comprehensive 72-lesson curriculum organized into modules covering AI platforms (ChatGPT, Claude, Gemini), creative tools (image, video, audio, design), coding and development, enterprise business applications, and AI leadership/governance.

## Lesson Files

All lessons are in this directory as individual markdown files: `lesson-00` through `lesson-72`.

## Module Structure

- **Lessons 00-09**: ChatGPT (tool selection, communication, writing, reasoning, data, images, voice, custom GPTs, API, enterprise)
- **Lessons 10-19**: Claude (fundamentals, research, writing, coding, computer use, projects, artifacts, Claude Code, agents, safety)
- **Lessons 20-28**: Gemini (fundamentals, multimodal, research, workspace, coding, thinking, long context, CLI, comparison)
- **Lessons 29-40**: Creative tools (image gen, Midjourney, Stable Diffusion, video, audio, design, video editing, 3D, presentations, personalization, photo editing, brand identity)
- **Lessons 41-51**: Coding and development (database, API, performance, security, DevOps, testing, code review, debugging, legacy, full stack, AI dev practices)
- **Lessons 51-60**: Enterprise business (sales, marketing, support, finance, product analytics, HR, legal, operations, integration)
- **Lessons 61-72**: AI leadership (teams, governance, change management, ROI, cost optimization, security, scaling, competitive advantage, future-proofing, ethics, case studies, trends)
