# Templates

Starter files for your vault. Copy these into your own vault folder, customise to your context, and use them as reusable scaffolds.

## Files in this directory

- `claude-md-template.md`, drop into your vault root as `CLAUDE.md`. Defines who you are, your voice rules, your never-dos, and your file structure for any AI working in your vault.
- `vault-structure/`, empty folder skeleton showing the recommended vault layout. Copy the folder names into your own vault.

## How to use these

The fastest path is to run the **Self-Onboarding Prompt** in `super-prompt.md` at the root of this repo. It will create the vault, the CLAUDE.md, and the folder structure for you, populated with real content from your inbox and calendar.

If you want to build by hand instead:

1. Create a vault folder somewhere on your machine (e.g. `~/[your-name]-brain/`)
2. Copy the contents of `vault-structure/` into it
3. Copy `claude-md-template.md` into the vault root as `CLAUDE.md` and fill in the bracketed fields
4. Initialise the vault as a git repo and push to a private GitHub repo
5. Start dropping notes into the relevant folders. The structure will help your AI navigate.
