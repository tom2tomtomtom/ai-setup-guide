## [0.1.0], 2026-05-27

Initial public release of the AI Setup Guide.

### Added
- Complete guide from Level 0 (mindset) to Level 4 (master operator), with Day 1 and Quickstart paths for beginners
- "Know your Claude" section mapping all surfaces, features, models, and plans, with concrete try-this examples per feature
- "Your first 7 days with Claude" day-by-day starter plan
- Skills deep dive with the full library (~180 skills grouped into 17 domain categories) and the weekly working set
- Coding protocols (Karpathy rules, plan/spec/execute, vault first, git as memory)
- New business workflows (BD Pulse, research dossiers, Hunter, AIDEN consult)
- The Self-Onboarding Prompt: a copy-paste prompt for Claude Code that interviews the reader, sweeps Gmail/Calendar/Drive, and bootstraps their vault
- Reference: full tool list (AIs, knowledge storage, comms, CLIs, weekly skills)
- Portable Setup Bundle documentation in `portable-setup/`
- Templates for vault structure and CLAUDE.md

### Notes
- Some operational details (VPS IP, personal file paths) have been scrubbed for public release
- Client names and project examples are retained as anecdotes (public information)
